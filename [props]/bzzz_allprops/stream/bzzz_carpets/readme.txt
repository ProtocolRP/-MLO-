Installation:
1) Insert folder "bzzz_carpets" to resources folder
2) Add to server.cfg
3) Restart server




--------------------------------------------------
If you stream props in another resource, you must edit the fxmanifest.
Add this line:
data_file 'DLC_ITYP_REQUEST' 'stream/bzzz_carpetpack.ytyp'

Then the server must be restarted. 
YTYP loads properties of props.
--------------------------------------------------

The props are ready for texture editing. 
You can modify the textures in OpenIV. 
Please adhere to the dimensions of 128x128, 256x256, 512x512...
