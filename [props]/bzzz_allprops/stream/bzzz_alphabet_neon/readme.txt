Installation:
1) Insert folder "bzzz_alphabet_neon" to resources folder
2) Add to server.cfg
3) Restart server


--------------------------------------------------
If you stream props in another resource, you must edit the fxmanifest.
Add this line:
data_file 'DLC_ITYP_REQUEST' 'stream/bzzz_neon_alph.ytyp'

Then the server must be restarted. 
YTYP loads properties of props.
--------------------------------------------------

Font: https://www.dafont.com/neon-sans.font

--------------------------------------------------
For CodeWalker users:
If you need to enlarge the prop, disable collisions in the ytyp file (change the flag to 0), 
then you can enlarge it in CodeWalker.
